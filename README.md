# nature作者学术中 release

这个公开仓库只用于 Zotero 插件自动更新。

它只包含：

- `nature-author-academic-center.xpi`
- `update.json`
- `update-beta.json`

这里不保存源码，也不保存 Supabase URL、Supabase key 或任何用户数据。

当前版本：`3.5.6`

首次从 `3.4.14` 切换到自动更新时，需要手动安装一次 `nature-author-academic-center.xpi`。之后 Zotero 的 `Check for Updates` 会读取本仓库的 `update.json`。
